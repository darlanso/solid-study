<h1> O Princípio de Substituição de Liskov (LSP)</h1>
<p>Devemos poder substituir classes base por suas classes derivadas em qualquer lugar, sem problema.</p>
<p><strong>Embora a herança favoreça o reaproveitamento de código, ela pode trazer efeitos colaterais quando não utilizada da maneira correta.</strong></p>
